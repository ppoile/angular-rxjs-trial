= Setup =

== Initial setup ==

```
$ nodeenv env --prebuilt
$ . env/bin/activate
$ npm install -g @angular/cli
$ ng new angular-rxjs-trial --defaults
```

== Checkout and... ==

```
$ git clone ...
$ cd angular-trial
$ nodeenv env --prebuilt
$ . env/bin/activate
$ npm install
```

= Development =

== Serve app ==

```
$ ng serve
```

== Test app ==

```
$ ng test
```

```
$ ng e2e
```
